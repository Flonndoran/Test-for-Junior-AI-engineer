# Финансовый AI-ассистент

Тестовое задание на позицию Junior AI-engineer.

## Описание

Консольный AI-ассистент для анализа финансовых данных компании за 20 лет (2005-2024). Ассистент:
- Загружает данные из CSV
- Рассчитывает рост выручки, операционную и чистую маржу
- Отвечает на вопросы пользователя, используя LLM
- Не придумывает числа, опирается только на предоставленные данные

## Архитектура

- `data_processor.py` — загрузка данных и расчет метрик
- `llm_client.py` — клиент для работы с LLM (поддержка OpenAI и Ollama)
- `main.py` — консольный интерфейс и цикл вопросов
