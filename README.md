# Финансовый AI-ассистент

Тестовое задание на позицию Junior AI-engineer.

## Описание проекта

Консольный AI-ассистент для анализа финансовых данных компании за 20 лет (2005-2024). Ассистент:
- Загружает данные из CSV файла
- Рассчитывает ключевые финансовые метрики (рост выручки, операционную и чистую маржу)
- Отвечает на вопросы пользователя, используя OpenAI API
- Не придумывает числовые значения, опирается только на предоставленные данные

## Требования

- Python 3.9 или выше
- API ключ OpenAI (получить на [platform.openai.com](https://platform.openai.com))
